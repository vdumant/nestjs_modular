## Description
[Nest Modular]
Generar modulo: nest g mo <module_name>
Administrar Variables de Ambiente: npm i @nestjs/config  |   npm i @nestjs/axios
Validar esquemas Env: npm i --save joi
Documentar Api: npm i --save @nestjs/swagger
Habilitar CORS: main.ts--> app.enableCors();
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
